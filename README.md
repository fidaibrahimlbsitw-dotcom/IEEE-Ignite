## IEEE Ignite

A responsive event website developed for IEEE SB LBSITW to showcase IEEE Ignite, a technical event focused on innovation, learning, and collaboration among students.

## Features

- Responsive design for desktop, tablet, and mobile
- Smooth scrolling navigation
- Home section with event details
- About IEEE section with useful links
- Event cards categorized into:
  - Technical
  - Professional
  - Interactive
- Search events by name
- Filter events by category
- Event schedule table
- Interactive FAQ section with expandable/collapsible answers
- Clean and modern user interface

## Built with:

- HTML5
- CSS3
- JavaScript
